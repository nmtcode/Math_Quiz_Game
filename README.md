# 🧮 Math Quiz Game (C++)

An interactive and educational command-line game designed to test and improve your mental math skills. This project features customizable difficulty levels, multiple operation types, and a structured round system.

---

## 🚀 Features
- **Customizable Rounds:** Choose to play from 1 to 10 rounds per session.
- **Multiple Operations:**
  - ➕ Addition
  - ➖ Subtraction
  - ✖️ Multiplication
  - ➗ Division
  - 🔀 Mixed (Random operations)
- **Difficulty Levels:**
  - 🌱 **Easy:** Simple numbers for beginners.
  - ⚡ **Medium:** Challenges your speed and accuracy.
  - 🔥 **Hard:** Complex calculations for math pros.
  - 🌀 **Mixed:** Randomly shifts difficulty each round.
- **Performance Summary:** A detailed final report showing your score and overall performance.

---

## 🛠️ Tech Stack
- **Language:** C++
- **Concepts used:** Structs, Enums, Random Number Generation, Input Validation.

---

## 🎮 How to Play
1. **Set the Stage:** Enter how many questions (rounds) you want to answer (1-10).
2. **Choose Level:** Select your preferred difficulty (Easy, Med, Hard, Mix).
3. **Choose Operation:** Pick the type of math problems you want to solve.
4. **Answer:** Solve the problems presented on the screen.
5. **Result:** Get instant feedback for each answer and a final score at the end.
